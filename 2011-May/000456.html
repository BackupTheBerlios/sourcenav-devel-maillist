Dear all,

I'm proud to announce that we've implemented a new feature to the xref
pane (thanks to Montaseri for suggesting it).
It is now possible to selectively remove subnodes based on their
"refers to" or "referred by" status from the xref graph.
Previously it was only possible to remove *all* subnodes which proved
to be not flexible enough for complex xref graphs.

I would like to specially thank Mark Thornber for writing itcl GUI
code (I never understand that) and pointing me in the right direction
on how to implement the feature - and for his gorgeous new icons
keeping the old SN spirit ;-)
The code has just been committed to trunk and will be contained in our
upcoming release NG4.4.
Please test!
