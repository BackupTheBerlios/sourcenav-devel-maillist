I would like to query the source database interactively, on a high level (SQL-like) if possible. Is there an easy way to quickly get answers to questions like:
* Are there class members whose name does not start with m_? I could use
  dbdump bin.iv | grep -v m_
  to get a rather exact answer.
* Does every method NewSetting call a function Refresh? In SQL I could do something like this:
  SELECT * from bin.md INNER JOIN bin.to 
  ON bin.md.class=bin.to.class AND
     bin.md.name=bin.to.symbol-name
  WHERE bin.md.name='NewSetting' AND bin.to.ref-symbol='Refresh'
to get all methods NewSetting calling Refresh and then compare with the list of all methods NewSetting (or even do that completely in SQL).

I was thinking of importing the btrees into another relational database that supports SQL. What other possibilities would exist?
