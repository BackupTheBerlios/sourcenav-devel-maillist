With the changes I've identified in the list already I have completed a 
build of the SVN code.  I used the following packages from 
https://sourceforge.net/project/showfiles.php?group_id=2435 to complete 
this build.

binutils-2.17.50-20060824-1.tar.gz
gcc-ada-3.4.5-20060117-3.tar.gz
gcc-core-3.4.5-20060117-3.tar.gz
gcc-g++-3.4.5-20060117-3.tar.gz
gcc-g77-3.4.5-20060117-3.tar.gz
gcc-java-3.4.5-20060117-3.tar.gz
gcc-objc-3.4.5-20060117-3.tar.gz
mingw-utils-0.3.tar.gz
mingwrt-3.15.1-mingw32.tar.gz
msysCORE-1.0.11-20080826.tar.gz
w32api-3.12-mingw32-dev.tar.gz

I created c:\snmingw
cd c:\snmingw
Then tar -zxvf each package.
I created a desktop shortcut with a Target of c:\snmingw\msys.bat and a 
Start in of c:\snmingw\bin.  You can use the icon in bin/rxvt.exe.

I create c:\opt\SN-trunk for the install prefix and configured the source

./configure --prefix=c:/opt/SN-trunk --enable-mingw --with-mingw -C


The changes are attached in a diff file.
