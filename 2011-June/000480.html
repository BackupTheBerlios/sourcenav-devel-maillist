Dear Gary, Rogerz,

thank you very much for your support. It's regarded warmly among us to
get "outsiders' help" so openly and quickly.

You can get SN NG by SVN:
$> svn co https://svn.berlios.de/svnroot/repos/sourcenav/tags/SN-NG4.4

or download a tar.bz2 for NG4.4 here:
http://developer.berlios.de/project/showfiles.php?group_id=8334

Building is usually done in the following way (the prefix shouldn't
matter much):

$> CFLAGS="-O2 -ffast-math -fomit-frame-pointer -march=i686"
./configure --prefix=/opt/SN-NG4.4 --enable-mingw -C
$> make && make install

I suspect we can get away with some of the old code by supplying a
"-std=gnu89" or something?
I don't have access to win32 machines on a regular basis, so it's
kinda hard to cross-check...

Cheers, and many thanks
~F
