> > I've built SN-NG4.3 for windows using mingw successfully. But after

> > So I did a directory compare and found out that some directories were not
> > included in SN-NG4.2 package, such as include, share\snavigator\demos,
> > share\sdk and some exe files (snpdbg.exe, snsdk.exe, tclsh83.exe,
> > tix4183.exe and wish83.exe).
> > I wonder if there's a file list for a windows binary distribution? or a
> > dedicated "make target" to create one?
>
> Currently, sadly, no. Contributions (a script or something) are of
> course most welcome.
>
> But you can safely delete all directories (and usually files, because
> we rarely add anything) that are not present in the distribution.
> Especially sdk/ and demos/ take up the most part.
> tclsh, tix and wish come from tcl/tk and tix, but are not needed to
> run sn (hyper includes those libraries directly).
>
> On *nix you could also remove lib/ , but you can't on win32.
