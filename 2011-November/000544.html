You may able to compile it mac os doing it as a unix program under 
X11,
 but in order to compile it as a native program you would need to port 
Tix
and update the autoconf scripts in the tcl and tk to acknowledge the
platform.

As for the changes from 8.4 to 8.5, the main problem is that Source
Navigator uses the internasl of the text widget in ways that were not
intended by the Tk programmers. In version 8.5 those internals changed
a lot so the port would be anything but trivial.
