I also backported the antialiasing font support for linux from Tcl/Tk 8.5, you
can see the results:

http://sourceforge.net/projects/getleftdown/files/SourceNavigator-Tryouts/SN-01.png/download
http://sourceforge.net/projects/getleftdown/files/SourceNavigator-Tryouts/SN-02.png/download

Since the benefits from antialiasing are arguable, it is not activated by
default, you need to configure SN with '--enable-xft'. You will probably also
need to install a libXft-devel package. In any case the end result will depend
on what fonts you have installed in your system as SN uses the Adobe family 
instead of the default Tk fonts.
