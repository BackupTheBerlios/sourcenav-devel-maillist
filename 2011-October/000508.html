ttk widgets are Themed Tk widgets, they were added at the Tcl/Tk core
for version 8.5 and are also available as an extension for 8.4.

Ttk uses native widgets in the different platforms, either Windows, with
vista, xp and winnative themes, macs and Linux.

You can get an idea of what it means to the end user with some screenshots
from a program of mine:

http://personal.telefonica.terra.es/web/getleft/snaps.html

Although those snaps have three advantages:

	- Tcl/Tk 8.5 which means antiased fronts.

	- It uses the tkpng extension so the icons are pngs not gifs.

	- It uses the TileQt theme which makes ttk use Qt widgets,
          unfortunately, it is no longer being maintained.

Porting SN to use them also means you can get rid of Tix and,
probably, incrTk.
