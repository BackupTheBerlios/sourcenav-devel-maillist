Hi,

I have ported version 4.5 to work with Tcl/Tk 8.4, although I must admit the
result is still alpha quality.

What I did was import the svn repository into github and then create a 
TclTk-8.4 branch to work in.

You can see github page at:

https://github.com/AndresGG/sn-8.4

There you will also find three files to download, two for windows:

https://github.com/downloads/AndresGG/sn-8.4/sourcenav-ng-setup-v4.5.80.exe
https://github.com/downloads/AndresGG/sn-8.4/SN-4.5.80-i686.tar.bz2

And the source tarball:

https://github.com/downloads/AndresGG/sn-8.4/sourcenavigator-NG4.5.80.tar.bz2

If you want to have antializing fonts in Linux you need to configure with the
'--enable-xft' flag and to have a libxft-devel package installed. The problem
with this is that source navigator will get some random font, eventually a
sane default will have to be chosen. Any suggestions?

Any comments welcome.

Andrés García
