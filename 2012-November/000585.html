Hi,

Sometimes installing all the legacy fonts your distro offers helps.

But if you are feeling adventorous, so may want to try a build with TrueType
support.

This is the way it looks in my Fedora system:

https://github.com/downloads/AndresGG/sn-8.4/Snap.png

If you want to try it, the source tarball is:

https://github.com/downloads/AndresGG/sn-8.4/sn-TruType.tar.bz2

To compile you need to configure it with '--enable-xft', but when you run it,
it will keep using the same fonts as the configuration is saved, so you may
have to delete your ~/.sn directory or change the font in the Preferences
dialog.
