Hi all,

I've just made a pre-release of a win32 build available for testing.
It's a minimal package (no docs, no man, no sdk) just intended for running SN.
As still win32 seems to be the cause for many problems please grab it
here and give it a workout:

http://developer.berlios.de/project/showfiles.php?group_id=8334

This is the first release with the "new" db4 (4.6.21) as database backend.
This means that you cannot use your existing project files (.proj) nor
your previous SN preferences - you will have to delete them and make a
new project as there is no binary compatibility between db-1.85 and
db-4.X.

Cheers,
Freek
