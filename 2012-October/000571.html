Dear all,

Source Navigator NG4.5 has been released. It is the continuation of
the very stable NG4 series and took just about 15 months of
development.
This time there wasn't a lot of change, but refinements to make usage
just a little better.


Fixes:

- Fix --noxref to actually not build and xref information
- Fix external_editor to deal with space in path under win32, part II
- Fix return code of main() call chain


Enhancements:

- Add new function to sn.el for direct jump to implementation in Emacs


Obtaining SN NG:

SVN people can switch to (or checkout) the following URL:
https://svn.berlios.de/svnroot/repos/sourcenav/tags/SN-NG4.5

Tarballs and binary builds may follow in the next couple of days here:
http://developer.berlios.de/project/showfiles.php?group_id=8334

As always - we're still looking for help and volunteers.

Cheers,
Freek
