Hi Medi,

> I was wondering if there is a way to set the new window size when creating a
> new windows for Edit (etc).
> For example from Window > New View > Editor, a window is created. However
> since I have a dual head configuration and my virtual screen size spans
> across both screens, the window is created with a width of 75% and

There already is a configuration option for that particular problem:

File -> Project preferences -> [Project] under Window:

    Window size is: xx % of screen size

Choose a xx of your liking, then click "Apply" and "Save as default"
(so it is valid as a general SN preference, too).

Other than that I assume that SN is a bit multi-head agnostic, esp. on
win32 due to tcl/tk's old age we have to use. As for any hooks - I
currently have no clue, but all the GUI handling is in
$SRC/snavigator/gui

Hope that helps and cheers,
F
