Hi Mark,

please find attached a patch for fixing a bug in our TK version
which causes the maximize button to be missing in KDE/KWin based
environments. It applies to NG4.2.

- Problem -
Running Source Navigator NG4.2 (and older for that matter) on a
recent KDE/Kwin based system (in my case an up to date Debian Lenny
on a i386 box), all Source Navigator NG windows have no "maximize"
title bar button; only "minimize" and "close" are available.

According to [1][2] the issue was fixed with TK 8.4.6, but given
Source Navigator NG brings it's own TK version (based on TK 8.3.2),
changes have to be applied accordingly.

- Solution -
The patch provided in the TK bug tracker [3] has been back-ported
in order to apply to the older TK version shipped and used by
Source Navigator NG.
