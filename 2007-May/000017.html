File snavigator/gui/execgrepdriver.tcl

Changes
1) Convert word boundary markers from egrep format to tcl format ('\<'
'\>' become '\m' '\M') when highlighting (tagging) matches.
2) Use filename+line_number pattern match on every line to determine
start position of search.
3) Determine length of matched pattern to use for highlighting (tagging).
4) Make default grep executable egrep.

Change 4 is a stop-gap.  Given the multiplicity of target systems and
of grep executables it may be better in the long run to ship a grep
with known behaviour as part of the distribution.  (The code in
execgrepdriver.tcl already supports this.)
