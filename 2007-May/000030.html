Ok - figured this one out...
implemented the call to gvim to have parameter +call cursor($line,$col)
which instructs gvim to open the file at the same position that the
cursor is at in editorpane.tcl
....
    set w $itk_component(editor)
    set line [${w} index insert]
....
        lappend external_editor "+call cursor($line,$col)"
