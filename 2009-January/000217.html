Hi Ryan,

not being one of the people doing the fork, but using and watching
the tool for some time the background looks like this:

The original Source Navigator tool went through many hands over the
years and finally ended on SorceForge. There it wasn't developed any
further, so the motivation for the fork was to restart the project.

As this Source Navigator NG has seen 3 releases since then.
The last one - Source Navigator NG3 - was then also published under
the name Source Navigator 6.0 (so different name only).

Best Regards,
Daniel


PS: Please note that the Source Navigator webpage [1] also points
     to Source Navigator NG for future development.
