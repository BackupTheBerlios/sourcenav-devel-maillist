Dear all,

I'm pleased to annouce an official release of Source Navigator 6.0 for win32.
Please note that you CANNOT use any 5.X parsed projects and have to
delete them first! This is due to the database library version change
that greatly improves performance.
You can get it here:

https://sourceforge.net/project/showfiles.php?group_id=51180&package_id=45043&release_id=627585


If you like to get the source code and more recent versions please
head over to the friendly Source Navigator NG project at:

http://sourcenav.berlios.de

We aim to improve performance and user experience (and also fix some
bugs). Our current version is NG3 and NG4 is in its final phase.

Cheers,
Freek
