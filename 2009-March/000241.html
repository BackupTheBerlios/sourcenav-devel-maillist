get the sources via svn:

   svn co https://lavape.svn.sourceforge.net/svnroot/lavape/trunk/lavape

http://www.qtsoftware.com/downloads before you can
successfully run "make" in the lavape root directory. The
above-mentioned example file LavaPE.cpp
resides in the src/LavaPE subdirectory, the "disturbing" moc-file in
its subdirectory "Generated".
(The moc files are generated from *.h files by Qt's moc precompiler.)
